# ChatGPT с нуля — STURZA LAB

Интерактивный бонус-диагностика перед практикумом «Сайт в GPT». Финальный экран включает безопасную для российского рынка карту публикации: ZIP из ChatGPT → клиентский аккаунт Beget → собственный домен и SSL. GitHub используется только как резервная копия.

Интерактивный бесплатный модуль для русскоязычных пользователей перед практикумом «Сайт в GPT: от идеи до ссылки».

## Что делает сайт

- определяет маршрут по текущей стране;
- для пользователей из России пошагово разбирает подключение, установку официального приложения и варианты оплаты своим платёжным способом из поддерживаемой страны;
- проверяет устройство, аккаунт, русский чат, голос и файлы;
- объясняет разницу между Free и Plus;
- не загружает выбранные пользователем файлы на сервер;
- сохраняет прогресс только локально в браузере;
- выдаёт персональный результат и предлагает практикум только после проверки готовности.

Россия не входит в официальный список поддерживаемых стран OpenAI. Сайт не обещает гарантированный доступ: пользователь видит предупреждение о риске блокировки, но получает практический маршрут вместо тупикового экрана.

## Публикация

GitHub Pages: Settings → Pages → Deploy from a branch → `main` / `(root)`.

После публикации сайт будет доступен по адресу:

`https://sturza-lab.github.io/chatgpt-start/`
